# ABC458 B - Count Adjacent Cells 题解

## 题意概括

给定一个 `H × W` 的网格。对于每个格子，求它有多少个上下左右相邻的格子。

## 解题思路

对每个格子 `(i, j)`，分别检查四个方向：

- 上：`i > 1`
- 下：`i < H`
- 左：`j > 1`
- 右：`j < W`

每满足一个条件，就说明这个方向存在一个边相邻格子，答案加一。

由于 `H, W <= 50`，直接双重循环枚举所有格子即可。

## 正确性说明

一个格子的边相邻格子只可能出现在上、下、左、右四个方向。

程序对这四个方向分别判断：

- 如果该方向没有越界，就说明对应格子存在，答案加一；
- 如果越界，就说明该方向不存在相邻格子，不加。

因此每个格子的计数恰好等于它的所有边相邻格子数量，算法正确。

## 复杂度

- 时间复杂度：`O(HW)`
- 空间复杂度：`O(1)`，不计输出所需空间

## 参考实现

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){

    int h, w;
    cin >> h >> w;

    for(int i=1; i<=h; i++){
        for(int j=1; j<=w; j++){
            int ans = 0;

            if(i > 1){
                ans++;
            }
            if(i < h){
                ans++;
            }
            if(j > 1){
                ans++;
            }
            if(j < w){
                ans++;
            }

            cout << ans << " \n"[j == w];
        }
    }

    return 0;
}
```
