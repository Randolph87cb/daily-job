# ABC458 C - C Stands for Center 题解

## 题意概括

给定一个只包含大写字母的字符串 `S`。统计有多少个子串满足：

- 长度为奇数；
- 正中间那个字符是 `C`。

相同内容但位置不同的子串要分别计数。

## 解题思路

把每个位置都看成可能的中心。

如果 `S[i] != 'C'`，那么以它为中心的合法子串数量是 `0`。  
如果 `S[i] == 'C'`，那么可以向左右同时扩展：

- 扩展 `0` 格，得到长度为 `1` 的子串；
- 扩展 `1` 格，得到长度为 `3` 的子串；
- 扩展 `2` 格，得到长度为 `5` 的子串；
- 依此类推。

最大能扩展多少格，取决于左右两边谁更短。因此这个位置的贡献就是：

`min(左边字符数, 右边字符数) + 1`

把所有 `C` 的贡献加起来就是答案。

## 正确性说明

固定某个位置 `i`：

- 若 `S[i] != 'C'`，则任何以 `i` 为中心的奇数长度子串都不满足条件，贡献为 `0`；
- 若 `S[i] == 'C'`，则每一个合法子串都唯一对应一个扩展半径 `r`，其中  
  `0 <= r <= min(i, n - 1 - i)`；
- 不同的 `r` 得到不同的子串，且所有满足条件、以 `i` 为中心的子串都会被枚举到。

因此位置 `i` 的贡献恰好是 `min(i, n - 1 - i) + 1`。  
把所有位置贡献相加，得到的就是全部合法子串数，算法正确。

## 复杂度

- 时间复杂度：`O(|S|)`
- 空间复杂度：`O(1)`

## 参考实现

```cpp
#include<bits/stdc++.h>
using namespace std;
using ll = long long;

int main(){

    string s;
    cin >> s;

    int n = (int)s.size();
    ll ans = 0;

    for(int i=0; i<n; i++){
        if(s[i] == 'C'){
            ans += min(i, n - 1 - i) + 1;
        }
    }

    cout << ans;

    return 0;
}
```
