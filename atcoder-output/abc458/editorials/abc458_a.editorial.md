# ABC458 A - Chompers 题解

## 题意概括

给定字符串 `S` 和整数 `N`。删掉 `S` 的前 `N` 个字符和后 `N` 个字符，输出剩下的部分。

## 解题思路

题目直接要求取中间一段。

删掉前后各 `N` 个字符后，保留下来的区间长度是 `|S| - 2N`，起点是下标 `N`。  
因此直接输出：

- 起始位置：`N`
- 长度：`|S| - 2N`

用 `substr` 就可以完成。

## 正确性说明

`substr(N, |S| - 2N)` 恰好从第 `N + 1` 个字符开始，连续取出 `|S| - 2N` 个字符。

因此：

- 前 `N` 个字符不会被取到；
- 后 `N` 个字符也不会被取到；
- 中间所有应该保留的字符都会被完整取出。

所以输出结果正是题目要求的字符串，算法正确。

## 复杂度

- 时间复杂度：`O(|S|)`
- 空间复杂度：`O(|S|)`

## 参考实现

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){

    string s;
    int n;
    cin >> s >> n;

    cout << s.substr(n, (int)s.size() - 2 * n);

    return 0;
}
```
