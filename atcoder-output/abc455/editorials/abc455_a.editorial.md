# ABC455 A - 455 题解

## 题意概括

给定整数 `A`、`B`、`C`。如果同时满足 `A != B` 和 `B == C`，输出 `Yes`；否则输出 `No`。

## 解题思路

直接按题意判断即可：

- `A != B` 表示前两个数不同；
- `B == C` 表示后两个数相同。

两个条件都成立时输出 `Yes`，否则输出 `No`。

## 正确性说明

程序恰好检查了题目要求的两个条件：

- 当 `A != B` 且 `B == C` 同时成立时，输出 `Yes`；
- 只要有任意一个条件不成立，输出 `No`。

因此输出结果与题意完全一致，算法正确。

## 复杂度

- 时间复杂度：`O(1)`
- 空间复杂度：`O(1)`

## 参考实现

```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){

    int a, b, c;
    cin >> a >> b >> c;

    if(a != b && b == c){
        cout << "Yes";
    } else {
        cout << "No";
    }

    return 0;
}
```
