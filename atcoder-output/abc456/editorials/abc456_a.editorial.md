# ABC456 A - Dice 题解

## 题目信息

- 原题链接：[A - Dice](https://atcoder.jp/contests/abc456/tasks/abc456_a)
- 题面对照：[../zh-CN/abc456_a.zh-CN.md](../zh-CN/abc456_a.zh-CN.md)
- 分值：100

## 题意概括

有三个标准六面骰子，每个骰子的点数范围都是 `1` 到 `6`。给定一个整数 `X`，判断三个骰子同时掷出后，点数和是否有可能恰好等于 `X`。

## 解题思路

三个骰子的点数和有明确的上下界：

- 最小值是 `1 + 1 + 1 = 3`
- 最大值是 `6 + 6 + 6 = 18`

所以答案其实只取决于 `X` 是否落在闭区间 `[3, 18]` 内：

- 如果 `3 <= X <= 18`，输出 `Yes`
- 否则输出 `No`

## 正确性说明

先看不可能的情况：

- 每个骰子至少为 `1`，所以总和不可能小于 `3`
- 每个骰子至多为 `6`，所以总和不可能大于 `18`

再看可能的情况：

对于任意 `X` 满足 `3 <= X <= 18`，令 `d = X - 3`，那么 `0 <= d <= 15`。  
我们先从 `(1, 1, 1)` 出发，总和为 `3`，再把这 `d` 点增量分配到三个骰子上。由于每个骰子最多还能增加 `5`，而总增量最多是 `15`，所以总能把 `d` 分配完，得到三个仍然落在 `1` 到 `6` 之间的点数。

因此，`X` 在 `[3, 18]` 内时一定可以构造出来；不在这个区间内时一定构造不出来。判定 `3 <= X <= 18` 即可。

## 复杂度

- 时间复杂度：`O(1)`
- 空间复杂度：`O(1)`

## 参考实现（Python）

```python
x = int(input())
print("Yes" if 3 <= x <= 18 else "No")
```
