# A - 3,2,1,GO

分值：$100$ 分

---

### Problem Statement

You are given a positive integer $N$.

给定一个正整数 $N$。

---

Output $N, N-1, \dots, 1$ in this order, separated by commas `,`.

按此顺序输出 $N, N-1, \dots, 1$，用逗号 `,` 分隔。

---

### Constraints

-   $1 \leq N \leq 9$
-   $N$ is an integer.

-   $1 \leq N \leq 9$
-   $N$ 是整数。

---

### Input

The input is given from Standard Input in the following format:

输入以如下格式从标准输入给出：

$N$

---

### Output

Output $N, N-1, \dots, 1$ in this order, separated by commas.

按此顺序输出 $N, N-1, \dots, 1$，用逗号分隔。

---

### Sample Input 1

```text
9
```

---

### Sample Output 1

```text
9,8,7,6,5,4,3,2,1
```

---

### Sample Input 2

```text
5
```

---

### Sample Output 2

```text
5,4,3,2,1
```

---

### Sample Input 3

```text
1
```

---

### Sample Output 3

```text
1
```
