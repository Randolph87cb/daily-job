# A - Closed interval

Source: https://atcoder.jp/contests/abc454/tasks/abc454_a?lang=en

Score : $100$ points

### Problem Statement

You are given integers $L$ and $R$.

Find how many integers are between $L$ and $R$, inclusive.

### Constraints

-   $1\le L\le R\le 100$
-   All input values are integers.

### Input

The input is given from Standard Input in the following format:

```text
$L$ $R$
```

### Output

Output the answer.

### Sample Input 1

```text
3 5
```

### Sample Output 1

```text
3
```

The integers between $3$ and $5$, inclusive, are $3,4,5$, which is three integers. Thus, output $3$.

### Sample Input 2

```text
1 7
```

### Sample Output 2

```text
7
```

### Sample Input 3

```text
14 79
```

### Sample Output 3

```text
66
```
