# A - Closed interval

分值：$100$ 分

---

### Problem Statement

You are given integers $L$ and $R$.

给定整数 $L$ 和 $R$。

---

Find how many integers are between $L$ and $R$, inclusive.

求 $L$ 到 $R$（包含两端）之间共有多少个整数。

---

### Constraints

-   $1\le L\le R\le 100$
-   All input values are integers.

- $1\le L\le R\le 100$
- 所有输入值均为整数。

---

### Input

The input is given from Standard Input in the following format:

输入按照以下格式从标准输入给出：

$L$ $R$

---

### Output

Output the answer.

输出答案。

---

### Sample Input 1

```text
3 5
```

---

### Sample Output 1

```text
3
```

---

The integers between $3$ and $5$, inclusive, are $3,4,5$, which is three integers. Thus, output $3$.

$3$ 到 $5$（包含两端）之间的整数为 $3,4,5$，共3个整数。因此输出 $3$。

---

### Sample Input 2

```text
1 7
```

---

### Sample Output 2

```text
7
```

---

### Sample Input 3

```text
14 79
```

---

### Sample Output 3

```text
66
```
