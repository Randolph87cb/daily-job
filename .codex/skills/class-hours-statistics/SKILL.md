# 课时统计

## 概览

- 优先运行 `scripts/generate_class_hours_statistics.py`，不要手工逐行抄表。
- 从一个或多个签到表读取指定月份课次，写入参考模板并保留模板样式。
- 先显式指定输入签到表、参考模板和输出文件，再核对结果文件。

## 默认规则

- 统一班级名称为 `CSP-J`、`进阶班`、`一对一`。
- 统一讲师姓名：`彬彬`、`曹彬` 都记为 `曹彬`。
- 把 `✅` 和 `请假已补` 都计入当次学员数量。
- `彭泓予`、`聂楚恒` 只有在该次课状态为 `✅`、实际到课时，才写 `2小时` 备注。
- 按 `班级 -> 原排课时段 -> 日期` 排序，让同一固定班次连续展示。
- 把 `2026/4/5调4/6` 这类表头按实际日期统计，但仍归入原排课班次分组。
- 当班级名称归一化为 `一对一` 时，把课程类别写成 `常规一对一课程`；其他班级写成 `常规班课`。

## 适用前提

- 在 Windows PowerShell 或任意可运行 Python 3 的环境执行。
- 已安装 `openpyxl`；缺失时先运行 `python -m pip install openpyxl`。
- 源工作簿第一页使用当前项目的签到表结构：
  - 含有 `上课日期` 行；
  - A 列是星期；
  - B 列包含班级和时间段；
  - C 列是讲师；
  - D 列是学生姓名；
  - H 列及之后是每次课的日期与状态。

如果源表不是这个结构，先停下并检查，不要直接套用脚本结果。

## 工作流

1. 确认参考模板和本月签到表路径。
2. 运行脚本生成新的课时统计文件。
3. 回读结果文件，重点核对班级分组、人数和备注列。
4. 如果当前项目维护了 `AI工作记录`，同步补充本次执行摘要。

## 运行方式

在项目根目录执行：

```powershell
python ".\\skills\\class-hours-statistics\\scripts\\generate_class_hours_statistics.py" `
  --month 4 `
  --reference ".\\课时统计\\课时统计表参考.xlsx" `
  --output ".\\课时统计\\26年\\4月\\4月课时统计.xlsx" `
  --sources ".\\课时统计\\26年\\4月\\4月金沙签到表.xlsx" ".\\课时统计\\26年\\4月\\4月文华签到表.xlsx"
```

如果输出文件正在被 Excel 占用，改一个新的输出文件名，或先关闭该文件再重试。

## 核对重点

- 同一个班级的固定班次已经连续展示，例如 `周三 18:30 进阶班`、`周五 18:30 进阶班`。
- `请假已补` 已计入人数。
- 只有 `彭泓予`、`聂楚恒` 实际到课的那几条记录，才带上对应的 `2小时` 备注。
- `Sheet2` 已写入规则说明、来源文件和生成时间。
