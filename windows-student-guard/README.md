# Windows 学生机设置脚本

这个目录提供一个适合学生机/机房机的基础 PowerShell 脚本，用来减少学生误改常见系统设置，并把鼠标与显示效果调整到更稳妥、易读的默认状态。

## 当前脚本会做什么

- 锁定任务栏，并禁止修改任务栏/开始菜单相关设置。
- 在 `Windows 10` 上把任务栏位置纠回到底部。
- 设置 `有外接鼠标时关闭触摸板`。
- 在 `Windows 10` 上把登录后的默认界面偏好设为桌面模式，并关闭自动弹出平板模式切换提示。
- 把鼠标速度恢复到 `Windows` 默认中间档（`10`），避免过快或过慢。
- 把鼠标指针切到 `Windows Black (large)`，兼顾可见性和不过度夸张的尺寸。
- 优先把主显示器切到显示器声明的推荐模式；如果拿不到推荐值，再回退到最高支持模式，优先保证清晰度。

## 不会做什么

- 不会删除应用、禁用系统服务、修改网络或安全策略。
- 不会在 `Windows 11` 上用非官方方式强改“平板模式”，因为微软已经移除了手动开关。
- 不会保证对第三方触摸板驱动 100% 生效；`Precision TouchPad` 设备最容易直接生效。
- 不会尝试修改 `Windows` 缩放比例；如果屏幕已经是高分辨率但仍嫌字小，应单独调显示缩放，而不是强行继续降分辨率。

## 运行方式

### 方式一：直接运行 PowerShell

```powershell
Set-ExecutionPolicy -Scope Process Bypass
.\Set-StudentSafeDefaults.ps1 -ApplyToDefaultUser -RestartExplorer
```

参数说明：

- `-ApplyToDefaultUser`：把同样的设置写入默认用户配置文件，后续新建学生账号时也会继承。
- `-RestartExplorer`：执行后重启资源管理器，让任务栏位置类修改立刻生效。
- `-MouseSpeed <1-20>`：自定义鼠标速度，默认 `10`。
- `-CursorScheme "<名称>"`：自定义鼠标指针方案，默认 `Windows Black (large)`。
- `-SkipDisplayResolution`：跳过显示分辨率调整。

### 方式二：双击命令包装器

直接运行 `Run-Set-StudentSafeDefaults.cmd`。

这个包装器会自动提权，并按“当前用户 + 默认用户模板 + 资源管理器重启”的完整基线执行。

## 变更记录

每次运行后，脚本会把本次改动前后的值写到 `backups/changes-时间戳.json`，便于后续人工回查。这个目录属于运行产物，不建议手工维护。

## 已知说明

- `Windows 11` 自身只支持任务栏停靠在底部，因此脚本只做锁定，不做位置强改。
- `Windows 10` 的任务栏“底部”设置通过 `StuckRects3` 注册表值实现，属于常见运维做法，不是微软主推的策略接口。
- 显示分辨率策略优先采用显示器声明的推荐模式；如果当前环境拿不到推荐值，再回退到最高支持模式，因为对 LCD/LED 屏幕来说，低于原生模式通常会变糊。
- 如果学生机是多用户环境，现有账号需要分别运行，或者由管理员结合用户登录脚本统一下发。
